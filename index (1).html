Cálculo diferencial — Optimización

El mínimo
matemático

Fundamentos, condiciones y métodos para encontrar el valor más pequeño que puede alcanzar una función.

f′(x₀) = 0  ∧  f″(x₀) > 0  ⟹  mínimo local en x₀
01 — Concepto

¿Qué es un mínimo?

Un mínimo es el punto donde una función alcanza su valor más bajo, ya sea dentro de un intervalo específico (local) o en todo su dominio (global).

Tipo A

Mínimo local

f(x₀) ≤ f(x) para todo x en algún entorno de x₀. La función es la más baja en esa vecindad, pero puede haber valores menores en otro lugar.

Tipo B

Mínimo global (absoluto)

f(x₀) ≤ f(x) para todo x en el dominio. Es el valor mínimo de toda la función, sin excepción.

02 — Condiciones

Cómo identificarlo

Para determinar si un punto crítico es un mínimo, se aplican las condiciones de primer y segundo orden.

Condición necesaria (1°):  f′(x₀) = 0
Condición suficiente (2°):  f″(x₀) > 0
f″(x₀) Clasificación Descripción
> 0Mínimo localLa función es cóncava hacia arriba en x₀
< 0Máximo localLa función es cóncava hacia abajo en x₀
= 0IndeterminadoSe requiere análisis de orden superior
03 — Método

Paso a paso

Para encontrar los mínimos de una función diferenciable f(x):

Calcular la primera derivada

Obtener f′(x). Esta describe la pendiente de la función en cada punto.

Igualar a cero

Resolver f′(x) = 0 para encontrar los puntos críticos x₀.

Calcular la segunda derivada

Obtener f″(x) y evaluarla en cada punto crítico.

Clasificar los puntos

Si f″(x₀) > 0, el punto es un mínimo local. Comparar todos los mínimos para identificar el global.

04 — Varias variables

Funciones de varias variables

Para f(x, y), el criterio se extiende usando el gradiente y la matriz Hessiana.

∇f(x₀, y₀) = 0  ⟹  punto crítico

H = | f_xx   f_xy |
    | f_yx   f_yy |

det(H) > 0 ∧ f_xx > 0  ⟹  mínimo local
Caso 1

det(H) > 0, f_xx > 0

Mínimo local. La superficie es cóncava hacia arriba en el punto crítico.

Caso 2

det(H) < 0

Punto de silla. No es mínimo ni máximo; la función sube en una dirección y baja en otra.

05 — Restricciones

Multiplicadores de Lagrange

Cuando se debe minimizar f(x, y) sujeto a una restricción g(x, y) = 0, se usa el método de Lagrange.

Minimizar f(x, y)   sujeto a   g(x, y) = 0

Condición:   ∇f = λ · ∇g
⟹ resolver el sistema con la restricción
